# Random Quote Generator App

A simple Android app built using Kotlin and Jetpack Compose.

## Features
- Displays motivational quotes
- Shows author name
- Generates a new random quote on button click
- Clean and simple UI

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## Output
Displays random motivational quotes with a "New Quote" button.
